CSS to the rescue

Dit is een vak over css en ik had gekozen voor de opdracht control panel.

Bij het vak CSS to the rescue konden we vanuit 4 verschillende opdrachten kiezen. Ik had gekozen voor de opdracht control panel. Het doel was om een control panel te maken dat gebruik maakt van CSS en het belangrijke is dat de gebruiker kan zien hoe het gebruikt moet worden, duidelijke feedback krijgt en de status van het systeem weergeeft. Verder mocht het niet meer dan 1 pagina zijn. Ook mochten we geen gebruik maken van classes en id's en op z'n minst 2 verschillende nieuwe css technieken gebruiken.
